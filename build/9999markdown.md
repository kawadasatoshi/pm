

## マークダウンでテーブルを書く方法

マークダウンの記法を用いることでhtmlのような複雑な手法を用いずとも直感的なテーブルを作成することが可能になります。





## 実際の書き方


## サンプルテーブル

```md

|  id    |  name   |  salary | dept_id |
| -----  |  ----   |  ----   | ----    |
|  1234  |  tarou  |  10,000 | 10      |
|  1235  |  kaiou  |  20,000 | 20      |

```


## 結果


|  id    |  name   |  salary | dept_id |
| -----  |  ----   |  ----   | ----    |
|  1234  |  tarou  |  10,000 | 10      |
|  1235  |  kaiou  |  20,000 | 20      |
